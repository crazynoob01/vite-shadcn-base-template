# Vite React Base Template

A minimal React + Vite starter with TypeScript, React Router, React Query, and Framer Motion. Styling is handled with plain CSS (no Tailwind or Radix UI).

## Features

- Vite for fast dev/build
- React + TypeScript
- React Router for routing
- React Query for data fetching
- Framer Motion for animations
- Custom CSS in `src/index.css` and `src/App.css`

## Tech Stack

- React
- Vite
- TypeScript
- React Router
- React Query
- Framer Motion

## Getting Started

### Prerequisites

- Node.js 18+

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app runs at `http://localhost:5173`.

### Production Build

```bash
npm run build
npm run preview
```
